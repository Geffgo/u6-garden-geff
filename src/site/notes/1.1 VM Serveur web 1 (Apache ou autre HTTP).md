# 1.1 VM Serveur web 1 (Apache ou autre HTTP)

> [!ATTENTION]
> La VM abrite plusieurs services vérifier que tout les autres services sont désactiver:
> - HaProxy
> - DHCP
> - DNS (si pas besoin)

### 1. Informations de la VM
- **Nom de la VM** : `VM-SERVEUR-GLPI`
- **OS** : Linux Debian 12 (64-bit)
- **Accès Système** :
    - **Utilisateur** : `sysadmin`        
    - **Mot de passe** : `netlab123`
- **Accès Web GLPI (par défaut)** : `glpi` / `glpi`

### 2. Finalisation (Interface Web)

Lien : `http://<IP_DE_VOTRE_SERVEUR>`

| **Paramètre**        | **Valeur**  |
| -------------------- | ----------- |
| **Serveur SQL**      | `localhost` |
| **Utilisateur SQL**  | `glpi_user` |
| **Mot de passe SQL** | `netlab123` |
| **Base de données**  | `glpi`      |

> 
> - **Admin** : `glpi` / `glpi`
> - **Technicien** : `tech` / `tech`
> -  **Utilisateur** : `normal` / `normal`
