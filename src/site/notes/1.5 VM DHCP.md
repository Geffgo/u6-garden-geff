# Fiche Technique : VM DHCP (VirtualBox)

> [!ATTENTION]
> La VM abrite plusieurs services vérifier que tout les autres services sont désactiver:
> - HaProxy
> - DNS (si pas besoin)
> - 
> Activer le service `isc-dhcp-serveur`avec la commande `systemctl start isc-dhcp-serveur.service`

## 1. Installation et Préparation

- **Installation :** `aptitude install isc-dhcp-server`
- **Vérification installation :** `dhcpd --version`
- **Interface d'écoute :** Modifier `/etc/default/isc-dhcp-server`
	- Supprimer une des deux `INTERFACES` et remplacer `INTERFACESvX`par`INTERFACES
    - `INTERFACES="eth0"` (remplacer par ton interface réseau).

---

## 2. Configuration du service (`/etc/dhcp/dhcpd.conf`)

### Options Globales


```
option domain-name "security-helpzone.lan";
option domain-name-servers 192.168.1.2, 192.168.1.3; # IPs de tes DNS
default-lease-time 86400; # 24h
max-lease-time 604800;    # 7 jours
authoritative;            # Serveur maître sur ce réseau
```

### Déclaration du sous-réseau (Subnet)


```
subnet 192.168.1.0 netmask 255.255.255.0 {
    range 192.168.1.100 192.168.1.200;  # Plage d'attribution (100 IPs)
    option subnet-mask 255.255.255.0;
    option routers 192.168.1.254;        # Passerelle par défaut
}
```

## 3. Administration et Tests

|**Action**|**Commande**|
|---|---|
|**Vérifier la syntaxe**|`dhcpd -t` (Crucial avant de relancer)|
|**Redémarrer le service**|`systemctl restart isc-dhcp-server`|
|**Vérifier le statut**|`systemctl status isc-dhcp-server`|
|**Voir les baux actifs**|`cat /var/lib/dhcp/dhcpd.leases`|
|**Logs (en direct)**|`tail -f /var/log/syslog`|

---

## 4. Diagnostic Client (Test de fonctionnement)

Depuis une autre VM cliente sur le même réseau :

1. **Demander une IP :** `dhclient eth0`
2. **Vérifier l'IP reçue :** `ip a`
3. **Simuler une requête (Audit) :** `nmap --script broadcast-dhcp-discover`

---

## 5. Gestion des Logs Dédiés (Optionnel)

Pour isoler les messages DHCP de syslog :

1. Dans `dhcpd.conf` : `log-facility local7;`
    
2. Dans `/etc/rsyslog.d/50-default.conf` : `local7.* /var/log/dhcpd.log`
    
3. Créer le fichier : `touch /var/log/dhcpd.log`
    
4. Relancer : `systemctl restart rsyslog isc-dhcp-server`
