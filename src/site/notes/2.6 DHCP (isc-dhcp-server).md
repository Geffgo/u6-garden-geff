m;# 2.6 DHCP (isc-dhcp-server)

### Configuration du service (`/etc/dhcp/dhcpd.conf`)

### Options Globales


```
option domain-name "security-helpzone.lan";
option domain-name-servers 192.168.1.2, 192.168.1.3; # IPs de tes DNS
default-lease-time 86400; # 24h
max-lease-time 604800;    # 7 jours
authoritative;            # Serveur maître sur ce réseau
```

### Déclaration du sous-réseau (Subnet)


```
subnet 192.168.1.0 netmask 255.255.255.0 {
    range 192.168.1.100 192.168.1.200;  # Plage d'attribution (100 IPs)
    option subnet-mask 255.255.255.0;
    option routers 192.168.1.254;        # Passerelle par défaut
}
```
