# 2.1 Configuration Réseau Linux

Cette fiche explique comment configurer une adresse IP statique via l'interface textuelle et comment tester la communication entre deux machines.

---

## 1. Modifier l'IP avec `nmtui`

L'outil **nmtui** (Network Manager Text User Interface) est la méthode la plus simple et visuelle en ligne de commande.

1. **Lancer l'utilitaire** :
    Bash
    ```
    nmtui
    ```

2. **Modifier la connexion** :
    - Sélectionnez **"Modifier une connexion"** (Edit a connection).        
    - Choisissez l'interface active (ex: `eth0` ou `ens33`).

3. **Paramétrer l'IPv4** :    
    - Passez la configuration IPv4 de `<Automatique>` à **`<Manuel>`**.
    - Sélectionnez **`<Afficher>`** à côté d'IPv4.
    - Remplissez les champs :
        - **Adresse** : `192.168.1.10/24` (par exemple).
        - **Passerelle** : L'IP de votre routeur.
        - **Serveurs DNS** : `8.8.8.8` (Google) ou l'IP de votre DNS local.
4. **Enregistrer** : Allez tout en bas et valider sur **`<OK>`**.
5. **Appliquer les changements** (Crucial) :
    - Revenez au menu principal.
    - Allez dans **"Activer une connexion"**.
    - Désactivez (Deactivate) puis réactivez (Activate) votre interface.        
    - Quittez l'outil.

## 2. Vérifier la connectivité (Ping)

Une fois l'IP configurée, il faut vérifier que les deux VMs se "voient".

- **Commande** :    
    Bash
    ```
    ping <IP_DE_LA_VM_CIBLE>
    ```

- **Interprétation** :
    
    - **Ça répond** : Vous voyez des lignes `64 bytes from... time=0.45ms`. La connexion est OK !
        
    - **Ça ne répond pas** : Message `Destination Host Unreachable` ou rien ne se passe.
